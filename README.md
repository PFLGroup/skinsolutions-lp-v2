## O projekcie
- Statyczna strona: `HTML + CSS + JS` (bez frameworka/build-stepu).
- Paleta: `#14130F`/`#000000` (czerń) · `#D2AC67` (złoto) · `#FFFFFF`/`#F3EEE6` (biel) · `#3a5d49` (zieleń marki — akcent).
- Mocne cechy układu (za wzorem): hero ze statami, „jak wygląda wizyta" (timeline), FAQ-akordeon, „zobacz nas", powtarzane/sticky CTA.
- Treść = realne dane klienta (https://skinsolutions.pl).

## Struktura
```

assets/css/styles.css   # system projektowy (luxe) + style
assets/js/main.js       # interakcje (nav, scroll-reveal, licznik, FAQ akordeon, sticky CTA)
```
